5-Minute Setup
You'll deploy a small Google Apps Script that bridges ClockWise and your Sheet. No coding needed — I've written it for you.
1
Go to sheets.google.com and create a new spreadsheet. Name it ClockWise Time Tracking.
2
Click Extensions → Apps Script. Delete any existing code in the editor.
3
Click ⬇ Download Apps Script below, open the file, copy all its contents, and paste into the Apps Script editor. Click Save (💾).
4
Click Deploy → New Deployment. Type: Web App. Execute as: Me. Who has access: Anyone. Click Deploy and authorize when prompted.
5
Copy the Web App URL shown after deployment. Paste it below and click Save & Test.
